1. Jelaskan menggunakan bahasa sendiri istilah Web, Portal dan Blog !

- Web (singkatan dari "World Wide Web") adalah sebuah sistem informasi global yang dapat diakses melalui internet. Ini terdiri dari berbagai halaman web yang terhubung satu sama lain melalui tautan hyperlink. Web digunakan untuk menyediakan berbagai jenis konten, informasi, aplikasi, dan layanan online, yang dapat diakses oleh pengguna melalui perangkat yang terhubung ke internet.

2. Jelaskan istilah Web Browser, berikan contoh !

- Web browser adalah perangkat lunak yang digunakan untuk menjelajahi internet dan menampilkan konten web. Browser membaca dokumen HTML yang diunduh dari server web dan menginterpretasikannya untuk menampilkan halaman web kepada pengguna. Contoh dari beberapa web browser yang populer termasuk Google Chrome, Mozilla Firefox, Microsoft Edge, dan Safari.

3. Apa yang dimaksud dengan Server Web, jelaskan !

- Server web adalah sebuah komputer atau perangkat keras yang menyimpan dan mengirimkan halaman web, gambar, video, dan berbagai jenis konten lainnya kepada perangkat pengguna melalui internet. Server web menjalankan perangkat lunak khusus yang disebut server HTTP (seperti Apache, Nginx, atau Microsoft IIS) untuk mengelola permintaan dari browser pengguna dan mengirimkan konten yang diminta. Server web berperan penting dalam infrastruktur internet untuk menyediakan akses ke situs web.

4. Jelaskan kerja browser dalam membaca script HTML !

- Ketika Anda memasukkan alamat URL atau mengklik tautan, web browser mengirimkan permintaan ke server web yang sesuai untuk mengunduh halaman web yang diinginkan.
- Setelah browser menerima dokumen HTML dari server, browser membaca dan menginterpretasikan kode HTML untuk membangun struktur halaman web.
- Browser mengidentifikasi elemen-elemen HTML seperti teks, gambar, tautan, dan lainnya, serta mengatur tampilan mereka sesuai dengan tata letak yang dijelaskan dalam kode HTML dan CSS.
- Jika halaman web mengandung script JavaScript, browser juga akan menjalankan script tersebut untuk menambahkan interaktivitas ke halaman.
- Hasil akhirnya adalah tampilan halaman web yang dapat dilihat dan digunakan oleh pengguna melalui antarmuka grafis yang diberikan oleh browser. Dalam proses ini, browser bertindak sebagai perantara antara server web yang menyediakan konten dan pengguna yang ingin mengakses dan berinteraksi dengan halaman web tersebut.
