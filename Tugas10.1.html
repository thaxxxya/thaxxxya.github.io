Jawaban

  1. Apa gunanya IFRAME?

    IFRAME (Inline Frame) adalah elemen HTML yang digunakan untuk menampilkan halaman web atau konten lain di dalam halaman web yang sedang dilihat. Ini memungkinkan pengembang web untuk menyisipkan halaman web atau media eksternal seperti video, gambar, atau dokumen dalam halaman utama. IFRAME berguna untuk menampilkan konten dari sumber eksternal di dalam halaman web tanpa mengarahkan pengguna ke halaman eksternal tersebut.

  2. Apa bedanya FRAME dan IFRAME?

    FRAME adalah elemen HTML yang digunakan untuk membagi halaman web menjadi beberapa frame atau jendela terpisah. Masing-masing frame dapat menampilkan halaman web yang berbeda dan berinteraksi satu sama lain. FRAME sering digunakan dalam konteks lama, tetapi sekarang kurang umum karena IFRAME lebih fleksibel dan aman. IFRAME adalah elemen HTML yang digunakan untuk menyisipkan satu halaman web atau konten eksternal di dalam halaman web utama. Ini tidak membagi halaman web menjadi frame terpisah seperti FRAME. IFRAME lebih fleksibel, lebih mudah dikendalikan, dan lebih aman dibandingkan dengan FRAME.

  3. Tuliskan bentuk umum perintah IFRAME secara lengkap:

                    <iframe src="URL_ke_halaman_web_eksternal" width="lebar" height="tinggi" title="Judul"></iframe>
